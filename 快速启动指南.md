# 🚀 Presentation AI 快速启动指南

## ✅ 已完成的配置

### Google OAuth 认证 (需要配置)
- **Client ID**: `需要配置Google OAuth Client ID`
- **Client Secret**: `需要配置Google OAuth Client Secret`
- **NextAuth Secret**: 需要生成安全密钥
- **重定向URL**: `http://localhost:3000/api/auth/callback/google`

## 🔧 剩余配置步骤

### 第一步：环境准备
确保已安装：
- [x] Node.js (需要修复当前环境问题)
- [ ] PostgreSQL 数据库
- [ ] 代码编辑器 (推荐 VS Code)

### 第二步：配置数据库
1. 启动 PostgreSQL 服务
2. 创建数据库：
```sql
CREATE DATABASE presentation_ai;
```

3. 更新 `.env.local` 中的 `DATABASE_URL`：
```env
DATABASE_URL="postgresql://您的用户名:您的密码@localhost:5432/presentation_ai"
```

### 第三步：配置AI服务 (可选)
获取以下API密钥以启用AI功能：
- **OpenAI API**: https://platform.openai.com/api-keys
- **Together AI**: https://together.ai/settings/api-keys

### 第四步：启动项目
```bash
# 1. 安装依赖 (如果Node.js环境正常)
npm install

# 2. 初始化数据库
npm run db:push

# 3. 启动开发服务器
npm run dev
```

### 第五步：测试登录
访问 http://localhost:3000 并使用Google账号登录

### Node.js 环境问题
如果您遇到Node.js错误：
1. 重新安装 Node.js (推荐使用 LTS 版本)
2. 或者使用 nvm 管理Node.js版本：
